# 📊 Análise de Desempenho de Alunos

## 🎯 Objetivo

Realizar uma análise exploratória de um dataset contendo informações de alunos (nome, idade, nota e status de aprovação), com o objetivo de identificar padrões de desempenho.

---

## 📂 Estrutura do Projeto

- data/: armazenamento dos dados
- notebooks/: análises realizadas
- README.md: documentação do projeto

---

## 🔍 Análises Realizadas

- Estatísticas descritivas das notas
- Distribuição de alunos aprovados e reprovados
- Análise básica entre idade e nota

---

## 🚀 Possíveis Evoluções

- Análise de correlação entre variáveis
- Criação de novas features (ex: faixa etária)


---

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- Matplotlib
