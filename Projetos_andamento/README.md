# 🎮 Análise de Dados - Jogos da Steam

## 🎯 Objetivo

Explorar dados relacionados a jogos da Steam para identificar padrões de mercado, comportamento de jogadores e possíveis fatores que influencam o sucesso de um jogo.

---

## 📊 Dados Utilizados

O dataset inclui informações como:

* Desenvolvedores
* Publicadoras
* Número de jogadores simultâneos
* Tempo médio de jogo
* Vendas (quando disponível)
* Popularidade dos jogos

---

## 🧠 Perguntas de Negócio

Este projeto busca responder perguntas como:

* Quais características estão associadas a jogos mais populares?
* Existe relação entre tempo de jogo e número de jogadores simultâneos?
* Quais desenvolvedores ou publicadoras têm maior destaque?
* Existe padrão entre preço, vendas e engajamento?

---
